
> Все вызовы выполняются **&НаКлиенте**.
> Асинхронность достигается описанием оповещения или использованием `Ждать …Асинх`.

---

## 1. Ввод числа `ПоказатьВводЧисла`

```1c
&НаКлиенте
Процедура ЗапроситьСумму(Команда)

    // «callback» сработает после закрытия окна
    Оповещение = Новый ОписаниеОповещения("ПослеВводаСуммы", ЭтотОбъект);

    //                                 длина 6, дробная 2
    ПоказатьВводЧисла(Оповещение, , "Укажите сумму заказа", 6, 2);

КонецПроцедуры

&НаКлиенте
Процедура ПослеВводаСуммы(Сумма, Параметры) Экспорт
    Если Сумма <> Неопределено Тогда
        Сообщить("Введено: " + Сумма);
    КонецЕсли;
КонецПроцедуры
```

## 3. Вопрос `ПоказатьВопрос`

```1c
&НаКлиенте
Процедура СпроситьЭкспорт(Команда)

    Оповещение = Новый ОписаниеОповещения("ПослеВопросЭкспорт", ЭтотОбъект);
    ПоказатьВопрос(
        Оповещение,
        "Выполнить экспорт данных?",
        РежимДиалогаВопрос.ДаНетОтмена,
        0,
        КодВозвратаДиалога.Нет,
        "Экспорт в XML");

КонецПроцедуры

&НаКлиенте
Процедура ПослеВопросЭкспорт(Ответ, Параметры) Экспорт
    Если Ответ = КодВозвратаДиалога.Да Тогда
        Сообщить("Экспорт запущен");
    КонецЕсли;
КонецПроцедуры
```

## 4. Асинхронный вопрос `ВопросАсинх`

```1c
&НаКлиенте
Асинх Процедура ЗадатьРежим()

    Текст = "Включить ночную тему интерфейса?";
    Кн = Новый СписокЗначений;
    Кн.Добавить(КодВозвратаДиалога.Да,      "Включить");
    Кн.Добавить(КодВозвратаДиалога.Отмена,  "Оставить текущую");

    Результат = Ждать ВопросАсинх(Текст, Кн);

    Если Результат = КодВозвратаДиалога.Да Тогда
        Сообщить("Ночная тема активирована");
    КонецЕсли;

КонецПроцедуры
```
